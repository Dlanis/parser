# parser
Parser for tl.rulate

Недавно залил более новую версию [RulateParser2](https://github.com/Dlanis/RulateParser2)

## usage
Скрипты протестированы только под линуксом, так что с виндой могут быть проблемы

`list.py` - парсит страницу тайтла, генерит лист глав

`parser.py` - парсит главы и генерит html
